# Sign-Language-Converter
A computer vision project that recognizes sign language gestures in real time. Built with Python, OpenCV, and TensorFlow — uses a custom-trained Teachable Machine model to classify hand signs through a live webcam feed.

Data Collection — Capture hand gesture images per letter using your webcam
Model Training — Train a classifier on Google Teachable Machine
Real-Time Recognition — Run the main script to detect and label signs live
